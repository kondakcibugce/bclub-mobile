# Buğçe Challenge Mobile

Bu proje Buğçe Challenge uygulamasının Expo ile çalışacak mobil sürümüdür.

## Expo Launch için

GitHub repo adresi:

```text
https://github.com/kondakcibugce/bclub-mobile
```

Expo Launch bu depoyu okuyup `package.json`, `app.json` ve `eas.json` dosyalarıyla mobil uygulama hazırlayabilir.

## İçerik güncelleme

Uygulamanın mevcut ekranları `bchallenge` klasöründedir. Tarif, video, koçluk, profil ve üyelik ekranları buradan yönetilir.

Expo uygulamasına aktarılan HTML dosyası şu komutla yenilenir:

```bash
npm run build:html
```
